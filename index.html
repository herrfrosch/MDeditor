What's MDedit?

MDedit is a free, open source markdown editor- special format of text, that's used for easy HTML/XHTML edition and creation. You can write completly new markdown file and open an existing one with MDedit. Exporting files with MDedit isn't a problem, you can export them in .md or .html file type.

Why MDedit?

There are plenty reasons to use MDedit but the most convicing are:
  • It's free- c'mon everyone loves that one.
  • It's open source- if you want to change something, just do it! Don't be shy check MDedit's github
  • You don't have to log in or download something. Pure online work.
  • It's mobile! You can use it anywhere in the world just with your smartphone
