What is emdeet?

Emdeet is online, easy-to-use, open source markdown format editor. It supports basic markdown syntax, also enables you to save your markdown files, online and on your device.

Getting started

Let's start working with emdeet by clicking editor link. After your browser load the page you will see similar image:

To start editing your text simply start typing and you will see text appering on both sides: left side is editor place, where you can enter your markdown text. Right side is diplaying styled text with principles of the markdown syntax.

Features

Formating text
If you want to format your text, you need to use basic markdown syntax. Emdeet supports it.
Saving file online
If you want to save your note online, first you need to accept emdeet's cookie policy. Then you need to add note title
and then click save button. After that popup, should appear with "saved sucessufly" text. After that you can close your browser, and when you again come to the emdeet's site, there's should be your note if you will open editor.
Acessing saved files
When you saved files online, you can acess them clicking in the library link. In the library, after clicking on the note on left side, it should appear on the main right window:
Using two buttons under preview window, you can open selected note in the editor or delete it.
Deleting files
As I mentioned above, you can delete note by selecting them in the library and clicking delete button. You can also delete current note in the editor by clicking delete button. It will delete note on which you are currently working.
Downloading files
You can download your file by clicking download button in the editor's menu. After that file should start downloading.
Uploading files
You can also upload your local file to the emdeet, by clicking upload button. This window should appear: Now click "choose file", and select in dialog window your markdown local file. After selecting file, click upload and your file should be uploaded to the editor
